# Storyboarding

Course Name: Critical Thinking, Design Thinking, Leadership and Teamwork

College Name: Galgotias University

Batch Number: 5

Task Number: 06

Task Name: Storyboarding

Topic: Integration of Design Thinking and Story Boarding

Product: Industrial Air Compressor Monitoring System

Description:
A 6-scene storyboard created for an Industrial Air 
Compressor Monitoring Startup. The storyboard shows 
the journey of Raj Sharma, a factory maintenance 
engineer, who faces unexpected machine failures and 
solves the problem using a real-time monitoring 
dashboard system.

Scenes Covered:
1. The Problem - Unexpected machine failure
2. Investigation - Finding root cause
3. Finding Solution - Discovering monitoring system
4. Using the Solution - Installing dashboard
5. Early Warning - Getting alerts before failure
6. Happy Outcome - Smooth production & cost savings
